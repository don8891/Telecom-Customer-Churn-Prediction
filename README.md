📊 Telecom Customer Churn Prediction

Machine Learning project to predict customer churn for a telecom company using synthetic data generation, exploratory data analysis, and classification modeling.

📌 Project Objective

Build an end-to-end machine learning system that:

Predicts whether a customer will churn in the next month

Provides probability scores for marketing prioritization

Identifies key drivers of churn

Meets performance criteria (Recall ≥ 75%, Precision ≥ 65%, ROC-AUC > 0.80)
